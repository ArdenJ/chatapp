# CHAT APP
## Premise 
A realtime chat app built using NodeJS Express, Apollo GQL, and React/NEXT.js

## Features 
- ### Authentication 
  - Login/Logout/register + session expiry
  - email verification
  - password reset
  - password confirmation
  - persistent login 
  - account lockout
  - rate limit (too mant requests: return 429)
- ### Chat
